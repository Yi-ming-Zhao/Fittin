# Fittin v1.0.9

这次更新完成了全应用主题系统与移动端自适应布局升级。

## 新功能

- 新增 5 套完整外观：黑曜黄铜、午夜钴蓝、波尔多丝绒、瓷白墨韵、浓缩余烬。
- 在「我的 → 外观」中直接预览并切换主题；选择会即时生效并在下次启动前恢复。
- Android、iOS 和 Web 冷启动使用与主题一致的预启动背景，避免先闪白或出现平台默认颜色。
- 背景、卡片、文字、线条、控件、底部导航、图表、训练反馈与身体解剖视图统一使用语义色，不再残留青绿色主题色。
- 保留奥林匹克杠铃片等具有固定现实含义的识别色，不随外观主题改变。

## 布局与体验

- 「今天」页按可用高度自适应：长屏使用舒展的顶部层级，短屏切换为自然的单列滚动，不遮挡底部导航。
- 「身体」页重新分配趋势图和指标卡空间；窄屏采用两列指标加一张通栏卡，宽屏使用三列布局。
- 训练记录卡片、杠铃视图、图表和导航在所有主题下保持一致的状态反馈与可读性。

## 质量验证

- 已覆盖 390x926 长屏、390x568 短屏，以及中英文移动端生产构建视觉矩阵。
- 已加入主题完整性、文字对比度、禁用青绿配色、主题持久化和自适应几何测试。

## Android 更新

- 已安装 v1.0.6 或更高版本的用户可以直接安装 v1.0.9 覆盖更新，训练数据与设置会保留。
- 如果设备仍停留在 v1.0.5 或更早版本，请先同步或备份训练数据，再卸载旧版并安装新版。
