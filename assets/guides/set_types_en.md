# Training Set Guide

Use these set categories to describe intent, not just reps.

## Straight Set
- Use when repeated work sets keep the same target style.
- Good for stable volume work such as 3x5 or 4x8.

## Top Set
- Use for the heaviest primary set of the exercise.
- Usually followed by backoff work or lighter straight sets.

## Backoff Set
- Use after a top set when load drops and volume continues.
- Good for accumulating work after the heaviest effort.

## AMRAP Set
- Use when the last set asks for as many reps as possible with control.
- Best when the program explicitly uses AMRAP for progression or effort checks.

## Warmup Set
- Use for ramp-up work before the main prescription.
- Warmup sets should not be used as the main working prescription.
