# Noto Sans SC

`NotoSansSC-Regular.ttf` is the regular Noto Sans SC build distributed by
Google Fonts. `NotoSansSC-AppSubset.ttf` is the runtime font: it contains the
code points used by Fittin's Dart copy, bundled plans and guides, plus Latin,
punctuation, arrows and full-width forms. Platform and Flutter font fallback
remain available for uncommon user-entered glyphs.

The curated subset keeps localized first-party screens available offline while
avoiding a 10.5 MB font download before Flutter Web's first frame.
